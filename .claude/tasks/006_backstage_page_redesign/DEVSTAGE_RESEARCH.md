## Key implications for agency page

1. **DevStage is "we", not "I".** It's a team practice (5 specialists) inside Żentała Innovation Agency. The agency Backstage page should NOT mix solo-Paweł voice with DevStage offering.
2. **DevStage already publishes a free 30-min consultation.** No invented "60-min discovery call" — match the real one.
3. **No published pricing.** No "€15k start" — DevStage frames it as "Pay for results, scale up or down."
4. **Roadie / Port.io are COMPETITORS, not Backstage versions we support.** My earlier line about "Spotify Portal, Roadie, Red Hat Developer Hub variants we track" was wrong on two counts: Roadie is a hosted Backstage competitor, and DevStage explicitly positions *against* them.
5. **Real differentiation already on devstage.io:**
   - Fast (cloud deploy in days)
   - No vendor lock-in / you own everything
   - 48h security SLA (concrete, hard to fake)
   - AI Chat with MCP Server (concrete product)
6. **Real proof:** 3 years, 4 enterprise implementations, 1 OSS plugin, 5 specialists. These are the numbers to use — not invented availability windows.
